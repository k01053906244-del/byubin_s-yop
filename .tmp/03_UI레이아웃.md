# [버스기사다~얍] UI 레이아웃 설계안

## 1. 메인 프레임워크 (Single Frame)
- **Header**: 좌측 상단 '버스기사다~얍' 로고, 우측 상단 현재 시간 (Lexend font).
- **Tab Content**: 5개 섹션 중 현재 활성 탭만 노출 (화면 전환은 탭 방식으로 처리).
- **Bottom Navigation**: 
    - [캘린더] (Active) - `/calendar`
    - [분석] (Layout Only) - `/analysis`
    - [팀원] (Inactive) - `/team`
    - [가이드] (Inactive) - `/guide`
    - [설정/점검] (Active) - `/manual`

## 2. 화면별 레이아웃 (Layout Details)

### Screen 1: 근무 캘린더 (활성)
- **Top**: '2024년 5월' 헤더.
- **Grid**: 7컬럼 달력 뷰.
- **Action Float**: 우측 하단 플로팅 버튼 영역
    - [AM] (Red Button): 오늘 날짜에 오전 근무 기록.
    - [PM] (Green Button): 오늘 날짜에 오후 근무 기록.
    - [전일] (Blue Button): 하루 종일 근무 기록.

### Screen 2: 근무 현황 및 분석 (부분 활성)
- **Top**: '배차표 입력' 큰 푸시 버튼. 
    - 클릭 시: "AI가 분석 중입니다..." 스피너 모달 노출.
- **Bottom**: 정적 이미지 또는 더미 그래프 (피로도, 누적 시간).

### Screen 3 & 4 (비활성)
- 상단 타이틀만 유지.
- 중앙에 `🚧 이 기능은 2단계 업데이트에서 준비 중입니다` 배너 노출.

### Screen 5: 운전자 매뉴얼 & 점검 리스트 (부분 활성)
- **Section 1**: [운전자 매뉴얼] PDF 아이콘 및 타이틀 (정적).
- **Section 2**: [운행 전 점검 리스트] (활성).
    - 리스트 아이템: 타이어, 냉각수, 오일, 라이트, 브레이크.
    - 각 아이템 우측에 대형 체크박스.
    - 전체 체크 시 하단에 "오늘도 안전운전!" 플로팅 메시지 노출.

## 3. 인터랙션 (Interactions)
- **탭 전환**: 가벼운 Opacity 전환 (애니메이션 최소화).
- **저장 알림**: 하단에 짧은 '저장되었습니다' 토스트 메시지.
