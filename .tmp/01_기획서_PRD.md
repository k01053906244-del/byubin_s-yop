# [버스기사다~얍] MVP 기획안 (Phase 1)

## 1. 핵심 요약
- **대표님께 드리는 이득**: 수동 배차 확인 및 점검 절차를 디지털화하여 **기사님의 업무 피로도를 낮추고, 운행 데이터의 실시간 자산화**를 위한 첫 단추를 채웁니다.
- **기술 용어 배제**: "기사님이 스마트폰 하나로 오늘 근무를 체크하고, 차량 안전 점검까지 10초 만에 끝낼 수 있는 솔루션입니다."

## 2. 사용자 경험 (UX)
- **근무 현황**: 기사님이 앱을 켜면 바로 이번 달 달력이 보입니다. 오늘 날짜를 누르고 '오전/오후' 버튼을 선택하면 즉시 저장되어 나중에 내 월급 계산 근거로 쓸 수 있습니다.
- **배차표 분석**: 복잡한 배차표 종이를 찍어서 올리는 대신, '배차표 입력' 버튼을 누르면 AI가 분석하는 연출이 나오고 2초 만에 자동으로 내 스케줄이 달력에 채워집니다 (1단계 목업).
- **안전 점검**: 운행 전 타이어, 냉각수 등을 손가락으로 가볍게 '체크'만 하면 "오늘도 안전운전!" 메시지와 함께 기록이 남습니다.

## 3. 리스크 알림
- **장비 최적화**: 8GB RAM 이하의 보급형 기기에서도 기사님이 끊김 없이 사용할 수 있도록 **초경량 설계(Internal Memory 최우선)**를 적용합니다.
- **사용자 적응**: 기존 종이 문화에 익숙한 기사님들을 위해 **폰트 크기와 버튼 반응성을 극대화**하여 디지털 전리에 대한 거부감을 최소화합니다.

## 4. 선택지 제안
- **A안 (최선책)**: 제안된 3가지 기능에 집중하고, 디자인은 StitchMCP의 'The Command Cockpit' 다크 모드 룩앤필을 적용하여 **프리미엄 전문가용 가시성** 확보. (강력 추천)
- **B안 (차선책)**: 캘린더 기능만 우선 활성화하고 나머지는 텍스트 매뉴얼로 대체. (시스템 부하는 낮으나 사용자 경험이 다소 떨어짐)

---

## [상세 요구사항 (PRD)]
- **기능 1 (Active)**: 캘린더 모듈 (AM/PM 상태 관리, LocalStorage 저장)
- **기능 2 (Active)**: 배차표 입력 목업 (버튼 클릭 시 모달 -> 2초 후 완료 메시지)
- **기능 3 (Active)**: 안전 점검 리스트 (체크박스 인터랙션, 올체크 시 축하 메시지)
- **비활성화 존**: 팀원 정보 및 노동법 가이드 (2단계 예고 텍스트 처리)
- **공통 UI**: 하단 고정 네비게이션 (캘린더/설정 이동 활성화)
- **성능**: JavaScript 프레임워크 지양, Vanilla JS/CSS 기반 최적화
